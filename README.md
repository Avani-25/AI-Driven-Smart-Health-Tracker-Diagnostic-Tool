# 🩺 AI-Driven Smart Health Tracker & Diagnostic Tool

> An AI-powered health monitoring platform that enables users to track vital health metrics, detect potential health risks, visualize trends, and receive personalized AI-driven health insights.
>
> ## 📌 Project Overview

The **AI-Driven Smart Health Tracker & Diagnostic Tool** is a modern healthcare web application that combines Artificial Intelligence, data visualization, and health analytics to help users monitor their well-being in real time.

The application allows users to securely manage their health profile, track important health metrics, visualize trends, and receive AI-generated recommendations based on their health data.

This project demonstrates the integration of Full Stack Development,Generative AI, and Health Analytics into a scalable healthcare platform.

## ✨ Features

-🧠 AI-powered clinical evaluation using Google Gemini AI
-❤️ Health risk prediction based on patient metrics
-📊 Interactive health dashboard
-📈 Statistical health score calculation
-⚠️ Critical clinical alerts generation
-🩸 Blood pressure analysis
-🍬 Blood glucose assessment
-😴 Sleep quality evaluation
-⚖️ BMI and lifestyle analysis
-📋 Personalized health recommendations
-🔒 Secure API key management using environment variables
-☁️ Cloud deployment on Render


## 🧠 AI Workflow
1.User enters health information.
2.Backend validates the input.
3.Health metrics are analyzed.
4.Patient data is securely sent to Google Gemini AI.
5.Gemini generates:
   -Clinical evaluation
   -Health risk assessment
6.Personalized recommendations
7.Results are displayed in an interactive dashboard.
